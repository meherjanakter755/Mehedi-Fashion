# 🛍️ Mehedi Fashion - আপনার ফ্যাশন স্টোর

একটি সম্পূর্ণ ফ্যাশন ই-কমার্স ওয়েবসাইট যা আপনি সহজেই পরিচালনা করতে পারেন।

---

## ⚡ দ্রুত শুরু (Quick Start)

### **1️⃣ লাইভ সাইট দেখুন:**
```
https://meherjanakter755.github.io/Mehedi-Fashion/
```

### **2️⃣ Admin Panel এ যান:**
সাইটে যান → **নিচে ডানদিকে ⚙️ Admin বাটন** ক্লিক করুন

### **3️⃣ এখানে করতে পারবেন:**
- ✅ পণ্য যোগ করুন
- ✅ পণ্যের দাম বদল করুন
- ✅ পণ্যের ছবি যোগ করুন
- ✅ WhatsApp নম্বর পরিবর্তন করুন
- ✅ দোকানের নাম পরিবর্তন করুন

### **4️⃣ সব পরিবর্তন সংরক্ষণ করুন:**
Admin Panel এর নিচে **💾 সব পরিবর্তন সংরক্ষণ করুন** বাটন ক্লিক করুন

---

## 📱 কীভাবে কাজ করে?

1. **Admin Panel** খুলুন
2. পণ্য যোগ করুন/সংশোধন করুন
3. **Save** বাটন ক্লিক করুন
4. ব্রাউজার রিফ্রেশ করুন - পরিবর্তন দেখতে পাবেন!

---

## 🔧 GitHub Pages এনাবল করুন (একবার)

যদি সাইট লাইভ না থাকে:

1. যান: [Settings → Pages](https://github.com/meherjanakter755/Mehedi-Fashion/settings/pages)
2. **Source** ড্রপডাউনে `main` branch সিলেক্ট করুন
3. **Save** করুন
4. 5-10 মিনিট অপেক্ষা করুন
